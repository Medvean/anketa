## Старт проекта

### Склонируйте репозиторий и перейдите в папку проекта
```
git clone https://github.com/Medvean/anketa.git
```

### Установите модули

```
npm i
```

или в разы в быстрее c [yarn](https://github.com/yarnpkg/yarn)

```
yarn install
```

### Запуск проекта
```
npm start
```

### Сборка в папку `dist`
```
npm run build
```
